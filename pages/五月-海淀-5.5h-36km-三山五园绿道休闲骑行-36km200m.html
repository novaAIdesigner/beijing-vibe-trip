路线摘要

按用户给的小红书帖重做：从火器营桥绿道入口接入，先经过北坞和玉泉片区，再一路向西到香山和西山森林公园。补给策略是“贴线小站密集布点”：火器营地铁口补水，中段北坞 / 闵庄路吃正餐或咖啡，西段香山买卖街和西山门口再补一次。

🏞️ 目的地介绍

🏞️ 三山五园绿道串联香山、玉泉山、万寿山等“三山”和清代皇家园林遗址。小红书参考帖的核心信息是：全程约 36km，火器营桥起步，大部分为绿道或低压力道路，沿途有休息点、卫生间、便利店、自动售货机和咖啡馆。它比“两山公园亲子线”更完整，也更适合做周末半日拉练。

✨ 活动介绍

🚲 推荐按 5 段骑：1 火器营桥地铁口补给起步；2 北坞公园 / 玉东两山田园段；3 船营公园 / 闵庄路骑行驿站段；4 香山买卖街咖啡 / 书店段；5 西山森林公园门口收尾。这样每 5-8km 都有一个可停靠点，不会出现“孩子饿了但还要离线找店”的问题。

✨ 活动节奏

☕ 最稳的吃喝节奏：火器营罗森 / 味多美先买水和面包；北坞午饭可选陶记、丹江渔村或金百万；闵庄路用林隙逸筑 / 四季乐咖做骑行咖啡站；香山段逛香山文化空间或雕刻时光；西山森林公园门口用香八拉徒步 NO.1 咖啡店收尾。

点评卡只放贴线锚点：地铁口、北坞 / 茶棚、闵庄路、船营公园、香山买卖街、西山森林公园门口；不推荐绕去万柳 / 远大路深处。

沿线站点与补给地图

段落站点 / 看点贴线补给怎么用
0 起点火器营桥绿道入口 / 火器营地铁站罗森(火器营地铁站)、味多美蛋糕(火器营店)、108融舍出发前买水、饭团、面包、能量小食；这里是最稳起步补给。
1 东段火器营 → 北坞公园沿线少量便利店；北坞后补给明显增多这段先热身，不建议一开始就绕路找店。
2 中段北坞公园 / 玉东 / 两山 / 玉泉借景分享咖啡、陶记(北坞店)、丹江渔村、金百万烤鸭第一次大休息。孩子饿了选陶记 / 金百万；想正式吃鱼选丹江渔村。
3 闵庄路船营公园 / 闵庄路 / 三山五园绿道中西连接林隙逸筑骑行驿站、四季乐咖骑行驿站、肯德基等快补最贴“骑行补给站”的段落，适合上厕所、补咖啡、检查车况。
4 西段香山公园 / 香山买卖街 / 香山文化空间香山文化空间艺书+西餐厅、雕刻时光、香山福宴这里可逛书店 / 文创 / 咖啡，但红叶季人多，要控制停留。
5 收尾西山森林公园门口香八拉徒步NO.1咖啡店、老马食府、杏林山庄餐厅大人完整骑行收尾点；孩子累了可到这里结束，不再硬返骑。

距离原则：优先 0-1km 内贴线补给。丹江渔村、金百万等虽不是绿道正中，但在北坞 / 茶棚 / 闵庄路片区，可作为“中途正餐”而不是远离路线的目的地。

亲子保守版：只骑火器营 → 北坞 → 玉东/两山 → 船营/闵庄路，约 15-20km 后撤。大人完整版再继续去香山 / 西山森林公园。

为什么这个方案值得保留

  • 用户给的小红书帖是明确路线帖，不是单点种草：起点、路线、全程、爬升、季节、装备、补给逻辑都可直接转化为执行方案。
  • 补给密度足够：火器营地铁口、北坞 / 茶棚、闵庄路、香山买卖街、西山森林公园门口，每段都有停靠点。
  • 点评真实商户能覆盖“便利店 / 面包 / 咖啡 / 骑行驿站 / 北京菜 / 书店西餐 / 西山门口快补”全链路。
  • 路线可以弹性截断：孩子状态差就中段撤，状态好再骑到香山和西山森林公园。

风险与临门一脚

  • 36km 对亲子不是低强度，默认给体力强孩子；低龄或天气热时必须改 15-20km 分段。
  • 香山 / 西山段节假日人流大，红叶季尤其不适合追速度。
  • 部分绿道与行人共享，必须礼让；路口下车推行。
  • 点评营业状态会变，特别是分享咖啡、骑行驿站、西山门口小店，出发前再点开原链复核。

复核入口：小红书《三山五园绿道休闲骑行》、点评“北坞公园 餐厅”、点评“香山公园 咖啡”、点评“西山森林公园 咖啡”。
