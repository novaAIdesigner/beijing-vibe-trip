怎么使用这个 skill

1

先说 vibe，不用先说地名。 比如“想找北京周边，轻徒步 + 咖啡 + 别太累的一日线”。

2

我会先发散 query，再去找候选。 先从小红书看灵感，再用六只脚 / 点评 / 携程 / Bing 交叉验证。

3

继续用这个 skill 时，我会默认怎么做

徒步类默认先查 六只脚，确认是不是成熟环线、里程和爬升是否靠谱；餐饮和咖啡优先补 大众点评；目的地公开存在性和外围攻略再用 携程 / Bing 兜底。默认从 五道口出发，自驾优先环线，穿越必须补接驳。

推荐提问方式

  • 继续给我找 3 条北京西边的一日线，优先咖啡收尾。
  • 把这条路线改成更适合亲子的版本，再补高分午餐。
  • 只保留六只脚成熟环线，别给穿越线。
  • 先看小红书 vibe，再用点评和携程确认值不值得去。
