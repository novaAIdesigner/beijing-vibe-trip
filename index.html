怎么使用这个 skill

1

先说 vibe，不用先说地名。 例如“北京周边，轻徒步 + 咖啡 + 别太累的一日线”。

2

我会先发散 query，再去找候选。 先从小红书看灵感，再用六只脚 / 点评 / 携程 / Bing 交叉验证。

3

最后收敛成能出发的路线。 默认会给你总时长、自驾 km、徒步参数、搭配商户 / 活动和风险点。

徒步类默认先查 六只脚；默认从 五道口出发，自驾优先环线，穿越线必须补接驳。
